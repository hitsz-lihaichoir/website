# 🏛️ 乐团组织架构

荔海合唱团隶属于哈尔滨工业大学（深圳），由指挥、声部长及各声部团员共同构成。

---

## 👥 组织架构

| 角色 | 职责 |
|------|------|
| **指挥** | 统筹排练、选曲与艺术指导 |
| **女高声部长** | 负责 Soprano 声部排练与考勤 |
| **女低声部长** | 负责 Alto 声部排练与考勤 |
| **男高声部长** | 负责 Tenor 声部排练与考勤 |
| **男低声部长** | 负责 Bass 声部排练与考勤 |
| **钢琴伴奏** | 排练与演出钢琴伴奏 |
